> *Real-time payment processing pipeline built for scale*

A production-grade monitoring system that ingests, validates, and processes high-volume financial transactions.

**Architecture Highlights:**
- 🟢 **Go** REST API for transaction ingestion & analytics
- 🦀 **Rust** gRPC processor for sub-ms validation
- 🐘 **PostgreSQL** for persistent transaction storage
- ⚛️ **React** dashboard for real-time analytics
- 📡 **gRPC + Protobuf** for inter-service communication

```
Client → Go API → gRPC → Rust Processor → PostgreSQL
                                ↓
                        React Dashboard ← Analytics API
```
